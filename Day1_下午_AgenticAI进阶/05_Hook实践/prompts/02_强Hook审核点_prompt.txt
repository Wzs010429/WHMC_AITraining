============================================================
强 Hook — 人工审核拦截 Prompt（7分钟，教师动手）
============================================================

输入 prompt（直接在 prompt 中指定审核点，不改 JSON）：
---
帮我做绘本：主题'登鹳雀楼'，面向小学四年级，6页。

审核点 1：大纲生成后必须停下等我确认，我确认后才能继续写文案。
审核点 2：所有内容写完后、最终网页生成前，必须停下等我确认。

每页都要配插图+语音旁白。
---

体验流程：
1. Agent 生成大纲 → 停！
   Agent: "大纲已生成，请确认：{...}"
   教师：确认，继续

2. Agent 自动跑完文案→插图→TTS → 停！
   Agent: "内容全部生成完毕，准备组装网页。请确认是否满意？"
   教师：第 3 页的背景太暗了，改成暖色调。确认后继续。

关键解说：
- "在 prompt 里直接写'审核点：...停下等我确认'——Agent 理解后主动暂停。"
- "JSON 配置的 Hook 是系统级拦截，Prompt 里的审核点是 Agent 行为级暂停。"
- "两种方式互补：全局规则用 JSON，任务级审核点写 prompt。"
