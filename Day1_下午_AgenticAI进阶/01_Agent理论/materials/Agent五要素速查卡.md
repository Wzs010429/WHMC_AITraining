# Agent 五要素速查卡（打印发给教师）

| 要素 | 比喻 | 绘本案例中的表现 | 一句话 |
|------|------|-----------------|--------|
| ① 模型 | 大脑 | DeepSeek V4 负责写文案、选情感标签 | "AI 的智力来源" |
| ② 工具 | 手 | 调图片 API（FLUX）、调 TTS（Higgs）、读写文件 | "AI 的手和脚" |
| ③ 流程 | 先后顺序 | 先规划→再写文案→再画图→再配音→最后组装 | "不按顺序就乱了" |
| ④ 记忆 | 笔记本 | 出第 5 页时记得第 1 页写了什么 | "AI 不会健忘" |
| ⑤ 人工审核 | 把关 | 大纲确认、最终预览，两个关键节点等你拍板 | "AI 干活，人把关" |

## LLM Loop

```
感知（读文件、理解需求）
  → 推理（分析问题、制定计划）
    → 行动（改文件、调工具、执行命令）
      → 反馈（检查结果、发现新问题）
        → 回到感知…
```

> "Agent 不是'答完就结束'，是'做完才结束'。"
