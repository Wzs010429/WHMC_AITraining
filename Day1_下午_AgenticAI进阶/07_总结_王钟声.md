### 三个核心原则（2min）

> 1. **Agent 不是全自动的** — Hook 让"人说了算"成为流程的一部分。AI 干活，人把关。
> 2. **Skill 封装专业判断** — 把好的做法写成菜谱，反复用、稳定用。你的教学经验变成 Skill，就是你的 AI 助手。
> 3. **MCP 即插即用** — 连接外部工具不需要从零写代码，统一标准一插就行。今天接的是图片和语音，明天可以接翻译、搜索、数据库。

### 衔接第二天

> "今天你们用 Agent 做了一个有声绘本。明天学什么？"
>
> - **明天上午**（Benchmark / 数据）— 怎么系统性评测绘本质量？一百个学生用了你的绘本，怎么知道好不好？
> - **明天下午**（多智能体 / RAG）— 多个 Agent 协作：一个 Agent 写文案、一个 Agent 画插图、一个 Agent 审核。加上知识库检索，就是一个完整的**智能课程系统**。
>
> "今天教会 AI 怎么做事。明天教会 AI 怎么把事做得更好。"
