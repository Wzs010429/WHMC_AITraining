# 产出清单 & 验证要点

## 每位教师应产出

- [ ] 1 个**带语音+插图的绘本网页**（浏览器可打开，翻页+播放）
- [ ] 1 个**自定义 Skill**（SKILL.md 文件，在 `.claude/skills/` 目录下）
- [ ] 至少 1 个 **Hook 配置**（settings.json 或 prompt 级别）
- [ ] **MCP 配置**：图片 MCP + TTS MCP 均已连通
- [ ] 能用自己话说出 **Agent 五要素 + LLM Loop**
- [ ] 会看 Claude Code 的**工具调用日志**

## Pulse Check（讲师中途检查点）

| 时机 | 检查方式 |
|------|---------|
| Agent 理论结束 → Harness | "Agent 五要素是哪五个？一起说——" |
| Skill Part1 开始前 | "环境都没问题的举下手？" |
| Skill Part2 → 休息 | "SKILL.md 文件生成成功的举下手？" |
| Hook → MCP | "settings.json 里加上 Hook 的举下手？" |
| MCP 连通验证后 | "图片和 TTS 都连通了的举下手？" |
