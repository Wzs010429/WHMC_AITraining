# Skill 创建检查清单

## 创建前
- [ ] 想好了 Skill 的功能——"输入什么，输出什么？"
- [ ] 想好了执行步骤——"分几步？每步做什么？"
- [ ] 选好了触发关键词——"说什么能触发这个 Skill？"

## 创建时
- [ ] 用精简模板描述需求（不要手写完整 SKILL.md，让 AI 帮你写）
- [ ] 执行步骤至少 3 步
- [ ] 输出格式明确写了要求（JSON / Markdown / 纯文本？）

## 创建后
- [ ] SKILL.md 文件已生成在 `.claude/skills/` 目录下
- [ ] Frontmatter（name、description、trigger）已填充
- [ ] 不满意就改："帮我把执行步骤第 X 步改一下"
- [ ] 测试：用一句话触发你的 Skill，看 Agent 是否按执行步骤走

> 不需要写代码——用自然语言写就行。你写得越清楚，它执行越稳定。
