# 教师创建 Skill — 6 个参考方向（含 Prompt）

## 精简模板（教师复制后填空）

```
帮我创建一个 Skill，叫____。
功能：____。
输入：____。
输出：____。
执行步骤：
1. ____
2. ____
3. ____
```

## 6 个参考方向

| 场景 | Skill 名称 | Prompt |
|------|-----------|--------|
| 论文批改 | 论文批改助手 | "帮我创建论文批改助手Skill。功能：输入学生论文→按评分标准打分→标出问题段落→给出修改建议。执行步骤：读取论文→逐项评分→标出问题→写修改建议→生成评语。" |
| 讨论题生成 | 讨论题设计师 | "帮我创建讨论题设计师Skill。功能：输入课程内容→生成3道开放式讨论题→每题附参考答案要点。执行步骤：分析课程内容→确定核心概念→生成讨论题→为每题写参考答案要点→附评分指引。" |
| 错题分析 | 错题分析师 | "帮我创建错题分析师Skill。功能：输入学生错题→分类错误类型→生成针对性练习。执行步骤：分析错题→归类错误类型→诊断错误原因→生成针对性练习→给出学习建议。" |
| 课程大纲 | 课程大纲规划师 | "帮我创建课程大纲规划师Skill。功能：输入课程名+课时→拆解章节→分配学时→标注重难点。执行步骤：分析课程目标→拆解章节→分配学时→标注重点难点→生成教学进度表。" |
| 行业简报 | 行业动态简报 | "帮我创建行业动态简报Skill。功能：输入行业关键词→搜索最新动态→汇总简报。执行步骤：分析关键词→收集最新动态→归纳趋势→生成简报→附信息来源。" |
| 教学案例 | 教学案例生成器 | "帮我创建教学案例生成器Skill。功能：输入知识点→生成3个生活化案例→附讨论问题。执行步骤：分析知识点→找生活关联→生成案例→为每个案例写讨论问题→检查案例是否覆盖核心概念。" |

> 关键提示：执行步骤是 Skill 的灵魂——想清楚你的任务分几步，每一步做什么。
