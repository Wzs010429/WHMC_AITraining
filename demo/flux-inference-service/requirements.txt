# 教学机环境：CUDA 12.6 + Anaconda/Miniconda
#
# ── 安装步骤（三步，全程国内镜像）─────────────────────────
#
#   1) 建一个全新的 conda 环境（Python 3.11）
#        conda create -n flux python=3.11 -y
#        conda activate flux
#
#   2) 一键安装本文件
#        pip install -r requirements.txt
#      （torch 自动走南大 cu126 源，其余包走清华源，见下方 index 指令）
#
#   3) 若第 2 步装的 diffusers 稳定版还不含 Flux2KleinPipeline，
#      改装 GitHub 主分支版（其余包已装好，这步只补 diffusers）：
#        pip install "git+https://github.com/huggingface/diffusers.git" \
#          -i https://pypi.tuna.tsinghua.edu.cn/simple
#
# ── 为什么 torch 不走清华源 ───────────────────────────────
#   清华 PyPI 镜像不提供带 CUDA 的 torch 轮子（只有 CPU 版）。
#   cu126 轮子改用南京大学镜像。CUDA 版本靠驱动向下兼容：
#   教学机是 CUDA 12.6，cu126 轮子正好匹配。
#
# ── 备用：torch 若下载慢，可换源（任选其一）──────────────
#   官方：  https://download.pytorch.org/whl/cu126
#   阿里云：https://mirrors.aliyun.com/pytorch-wheels/cu126
# ============================================================
